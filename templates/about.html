How the Chatbot works?

The database is from kaggle, which contains basic questions about computer science. I trained the dataset with a simple DNN model. The chatbot answers from the trained data, and if an unknown question is asked, it learns from the user. The newly learnt answers are stored in a separate JSON file along with the question. Next time a user asks a question, the bot tries to get answers from both the training data and newly created JSON. The bot matches with the closest answer and sends it to the front end.

Features of Chatbot

  • Answers basic questions related to computer science.
  • Model has about 82% accuracy.
  • Can learn from user, if the chatbot does not know.
  • Used natural language processing concepts like: lemmatization and bag of words.

Challenges faced while developing

  • Faced difficulty in creating the teaching logic using JS.
  • Opted to use JS for the logic as it would be far simpler to develop than having the logic in app.py
